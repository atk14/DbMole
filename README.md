DbMole
=====

DbMole provides basic functionality with databases.


Installation
------------

Just use the Composer:

    composer require atk14/dbmole

Testing
-------

Install required dependencies for development:

    composer update --dev

Run tests:

    ./vendor/bin/phpunit
